# CodeAlpha_HotelReservationSystem


## Description
This project is a console-based Hotel Reservation System developed using Java for the CodeAlpha Internship.

The application allows users to:
- View hotel rooms
- Book rooms
- Cancel bookings
- View room availability

The system uses Object-Oriented Programming concepts and menu-driven interaction.

## Features
- Room categorization
- Room booking
- Booking cancellation
- Payment simulation
- Availability checking
- Console-based interface

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## How to Run

Compile:
javac HotelReservationSystem.java

Run:
java HotelReservationSystem
